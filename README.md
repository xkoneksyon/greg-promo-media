# Greg Promo Media - Premium Static Website

This is a high-end static website for Greg Promo Media and the Haitian All Star Soccer Game. It is ready for GitHub Pages.

## Files

- `index.html` - full one-page website
- `styles.css` - premium responsive design
- `script.js` - menu, animation, countdown, and email form behavior
- `assets/greg-profile.jpeg` - profile image
- `assets/haitian-all-star-2026.jpeg` - event flyer

## Publish on GitHub Pages

1. Create a new GitHub repository named `greg-promo-media`.
2. Upload all files from this folder to the repository root.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch: `main`, folder: `/root`.
6. Click **Save**.
7. GitHub will publish the site at `https://YOUR-USERNAME.github.io/greg-promo-media/`.

## Customize Before Launch

- Replace `booking@gregpromo.com` with the official booking email.
- Replace the Eventbrite link if the ticket link changes.
- Add official sponsor logos inside the sponsor section.
- Connect the contact form to Formspree, Netlify Forms, or a backend when ready.
- Buy a custom domain such as `gregpromo.com`, then connect it in GitHub Pages.
