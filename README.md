# Greg Promo Media — Premium Website v3

This is a premium GitHub Pages-ready website package for Greg Promo Media.

## Included
- `index.html`
- `styles.css`
- `script.js`
- `404.html`
- `assets/`

## Features
- High-end editorial look inspired by premium media websites
- Light mode + dark mode toggle
- Mobile responsive layout
- Greg Promo branding and Instagram link
- Haitian All-Star Soccer Game section with countdown and ticket link
- Contact / inquiry form

## GitHub Pages upload structure
Upload these at the **root** of your repository:

```
assets/
index.html
styles.css
script.js
404.html
README.md
```

## Publish on GitHub Pages
1. Go to **Settings → Pages**
2. Choose **Deploy from a branch**
3. Select **main** and **/root**
4. Save

Then open:

`https://xkoneksyon.github.io/greg-promo-media/`

## Important
If you already uploaded older files, replace them with these new ones.
